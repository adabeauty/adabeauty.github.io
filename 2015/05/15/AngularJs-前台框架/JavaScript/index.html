AngularJs-前台框架

By Huan Wang

發表於 2015-05-15

定义

AngularJs 是一套由Google打造的JavaScript前端框架，对于动态网页的开发尤其适合。
做过 JS 开发的人都会知道，对于用 DOM 或者 JQuery 动态操纵view上的元素是
一件比较麻烦的事情。但是当使用了 AngularJs 之后，这个问题便迎刃而解。

特性

  1. MVC
    MVC 是模型(model)－视图(view)－控制器(controller)的缩写，使用一种业务逻辑、
    数据、界面显示分离的方法来组织代码。其中：
    Model ：就是应用中的数据，一般式 JavaScript 对象。
    View ：是AngularJs 解析后渲染和数据绑定后生成的HTML。
    Controller : 可以初始化初始化页面对象，并且定义和实现一些 View 上需要的方法。

  2. 数据双向绑定
    通过 $scope 是 AngularJS 中的一个 JavaScript 对象， 可以完成数据绑定的任务。
    在 Controller 中使用 $scope 可以针对特定的 View 来扩展数据和特定的功能。无论是
    页面上被绑定的元素还是 controller 中的 $scope 的数据值发生了变化，另一方的值也会
    发生变化。

使用方法

  1. 建立index.html文件
    在 body 标签上其中使用 ng-app ，给出 app 的名称，然后在 body
    内的 div 上使用 ng-view=””, 最后引入所有需要的 js 文件（包括一些库函数和controller文件）。

  2. 建立app.js
    使用 var app = angular.module('myApp', [ 'ngResource','ngRoute','checklist-model']);，声明 module 的名称（此处声明为MyApp）。然后使用.config 和 .when来配置模板的 url 和 controller。

更多详情：请参考AngularJs官网
