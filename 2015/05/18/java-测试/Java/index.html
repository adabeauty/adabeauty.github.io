hamcrest 断言框架

hamcrest 源于 java， 但现在可支持多种语言，
如 Phyton、Ruby、Objective-C、PHP、Erlang 和 Swift。
最大的特点是它的匹配器， 可以自由组合创建灵活的表达式， 也可以自定义你所需要的各种匹配器，从而满足
不同测试的需要。

Mock 框架 - mockito

mockito 是 java 单元测试的 mock 框架。mockito 最大的价值在于
你可以以可读性更高的方式书写 mock、stub、spy方法，而不用记住这些容易让人困惑的概念。
而且 mockito 能够记录所有被 mock 函数的交互（包括是否被调用、调用几次、使用什么参数调用等等）其功能
如下所示：

  1. 使用 mockito 验证行为。

  2. 使方法调用返回特定期望的值，也称 stubbing.(默认情况下， 所有有返回值但是没有被 stub 的方法都会返回默认值， 如：
    int/Integer 类型 会返回0， boolean/ Boolean 类型返回false， 其他类型则会返回 null)。

  3. 参数匹配

  4. 验证方法是否被调用，或者被调用的特定的次数。

  5. 调用无返回值的方法时， 返回异常。（使用 doThrow）

  1. 验证函数被调用顺序

更多功能参考 mockito 官网.

如果使用 maven 或者 gradle 进行管理时， 可选择到mvnRepository
