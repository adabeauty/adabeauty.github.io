Introduction to Hexo

It’s A fast, simple & powerful blog framework which is based on Node.js.

You can write posts in Markdown (or other languages) and Hexo generates static files with a beautiful theme in seconds.

Traits

Blazing Fast

Node.js brings you incredible generating speed. Hundreds of files take only seconds to build.

Markdown Support

You can use Markdown to write the blog.

A brief usage of Markdown.

One-Command Deployment

You only need one command to deploy your site to GitHub Pages, Heroku(A Cloud Application Platform) or other sites.

Various Plugins

Hexo has a powerful plugin system. You can install more plugins for Jade, CoffeeScript plugins.

There are three requirements if we want to use Hexo.

How to write my own blog

create a draft

When you want to write a new blog, you can create a draft file first. You can execute the following command and then hexo can generate a file named use-Hexo-to-build-a-blog.md in source/_draft.

edit file

In this step, you can choose a editor(I recommend to use Atom which can make us to view the visible effect of current file at anytime with shortcut(shift + ctr + M)) to edit the draft file by yourself.

When you finish the edition of the file, it’s time to publish and then you can find this file in _post.

Just using the name of file(don’t add suffix) to deploy is ok.

ps: All of files in the _post folder can’t be deploy to our website. Only the files in _post can be posted to our repository and deployed.
