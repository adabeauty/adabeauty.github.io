多态

那么什么是多态呢？曾经听有人用一句很经典的话总结了一下：多态就是子类的对象可以赋给父类的引用。虽然这句话也是他从书上看来的，并不是他的原创。但是这句话我认为对多态确实描述的比较准确。

再举个例子，如果你想去宠物店买一只宠物，你可能会给店员说我想买一只小动物，那么店员可以给你一只小猫，小狗或者兔子。因为他们都是动物呀。这就说明这个动物具有多态性，它具有多种表现形式，可以使是小猫，小狗或者兔子。
