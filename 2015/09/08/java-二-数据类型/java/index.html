在java中数据类型可分为两大类，分别是基本类型和引用类型。

primitive 基本数据类型

包含类型

基本特点

对于基本数据类型而言，它保存的是基本类型的值。

引用数据类型

包含类型

总体来说对象和数组都是引用类型。具体包含的类型如下：

Java封装类

java的封装类都是引用类型，例如Integer、Double、Float、Byte等。

对象

所有的对象都是引用类型，它存储的是该对象在内存当中的地址。

要求java虚拟机分配栈空间给引用变量，并将这个变量命名为myDog。

数组

数组永远是引用类型，但是它其中存储的元素本身可以是primitive类型或者引用类型均可。

基本特点

对于引用数据类型而言，它最基本的特点就是存储的不是数据本身，而是数据在内存中的地址。然后java虚拟机知道如何根据这个地址去找到该数据本身。
