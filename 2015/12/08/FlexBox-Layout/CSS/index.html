Introduction

flexbox是css灵活盒布局模块，旨在提供一种更加高效的方法来布局、分配容器中的各个item之间的空间，即使他们的大小是未知或者动态的（因此才使用flex，意为灵活的）。

其核心四星就是赋予容器去更改它的每个item的宽/高从而最好的填充可用空间（大多时候都是为了适应各种各样的显示设备和屏幕的大小）。一个灵活的
容器可以扩展它的item去填充可用空间，也可以通过shrink的方式来防止越界。

和普通的布局不同的是，flex与方向无关（block是垂直相关的，而inline则是水平相关的）。这些普通的布局对于页面来说，可以正常工作，但是他们缺乏灵活性，从而很难支持大而复杂的应用，尤其是当定向发生变化、改变大小、拉伸或者缩进等等的时候。

Note: FlexBox 布局尤其适用于应用中的component和小规模布局。而Grid想要做的是更大规模的布局。

基本使用

parent(flex container) 的属性

  1. display
  2. flex-direction
  3. flex-wrap
  4. flex-flow
  5. justify-content
  6. align-items
  7. align-content

children(flex items) 的属性

  1. order
  2. flex-grow
  3. flex-shrink
  4. flex-basis
  5. flex
  6. align-self
