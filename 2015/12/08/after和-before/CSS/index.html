介绍

::after和::before实际都是伪元素，它们允许我们从无需在HTML中修改，便可通过CSS中向页面插入内容但却不会影响已有的DOM。
使用::after和::before可在已有的DOM元素上之后或者之前添加新内容。

  • E:before/E::before设置在对象前（依据对象树的逻辑结构）发生的内容。用来和content属性一起使用
  • E:after/E::after设置在对象后（依据对象树的逻辑结构）发生的内容。用来和content属性一起使用

content 的值类型

对于content的值可以是以下四种类型：

  1. String: content: "a string"; 对于特殊字符需要使用转义字符（unicode entity）。
  2. Image: content: url(/path/to/image.jpg);图片插入的尺寸就是它的确切尺寸，而且它的大小是不能改变的。
  3. Nothing：content: ""; 这种方式对于clearfix清除浮动或者插入background-images 背景图片是非常有用的，而且这种情况下的图片大小是可以改变的，可通过设置高和宽或者background-size来改变图片大小 。
  4. A counter：content: counter(li); 配合marker使用对于styling list来说非常有用。

    Notes: 需要注意的是content中是不可以插入HTML的

: VS ::

首先所有支持::的浏览器均支持:语法，但是只有IE8只支持:。所以说为了提供对各种浏览器的更好支持，推荐使用:。

::是一种用来从伪元素选择器中辨别为元素内容的新格式，只要不使用IE8，那么你就可以放心的使用::了。

浏览器支持情况

  • Firefox 3.5- 不允许为元素使用absolute positioning
  • 对于In Opera 9.2，在伪元素内部，空格总会显示出来
  • IE 8 不支持在伪元素上使用z-index
